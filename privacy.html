와이투두(Y2do, 이하 "회사")는 잇데이(Itdasy, 이하 "서비스") 사용자의 개인정보를 중요하게 여기며, 「개인정보 보호법」 및 관련 법령이 정한 바를 준수하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

1. 처리하는 개인정보 항목 및 목적

필수이메일, 비밀번호(암호화), 이름회원 식별·로그인·고객센터 응대회원 탈퇴 시까지

자동 수집접속 IP, 디바이스 정보(모델·OS), 사용 로그, 쿠키보안 (로그인 시도 제한), 서비스 품질 개선1년 (통신비밀보호법)

2-1. 처리 위탁

위탁받는 자위탁 업무위탁 정보

2-2. 국외 이전 동의

회원가입 시 "개인정보 국외 이전에 동의합니다" 체크박스를 통해 동의를 받습니다. 미동의 시 서비스 제공이 불가능하며, 추후 설정 → 계정 → 국외이전 동의 철회에서 동의를 철회할 수 있습니다. 철회 시 Gemini 기반 AI 기능(캡션·비서)은 사용 불가합니다.

2-3. 가명처리(Pseudonymization)

AI 분석 시 고객 개인정보는 서버에서 가명(예: "고객#1", "고객#2")으로 치환된 후 외부 AI 서비스에 전달됩니다. 실제 이름·전화번호 등 식별 정보는 Google 등 외부 서버에 저장되지 않습니다. Google Gemini API 정책상 입력 데이터는 모델 학습에 사용되지 않습니다.

3. 개인정보의 보유·이용 기간 및 파기

  • 회원이 회원 탈퇴를 요청하면 즉시 모든 개인정보를 파기합니다(관계 법령에 따라 보관이 필요한 경우 예외).
  • 법령에 따른 최소 보관:
    • 계약·청약철회 기록: 5년 (전자상거래법)
    • 대금결제·재화공급 기록: 5년 (전자상거래법)
    • 소비자 불만·분쟁 처리 기록: 3년 (전자상거래법)
    • 로그인 기록: 3개월 (통신비밀보호법)
  • 파기 방법: 전자적 파일은 복구 불가능한 방법으로 삭제, 출력물은 분쇄합니다.

4. 정보주체의 권리·의무 및 행사 방법

이용자(회원·고객)는 언제든지 다음 권리를 행사할 수 있습니다:

  • 개인정보 열람·정정·삭제·처리 정지 요구
  • 개인정보 처리 동의 철회
  • 앱 내 설정 → 계정 → 데이터 내보내기 (CSV 다운로드)
  • 권리 행사: contact@itdasy.com 로 이메일 요청 → 7일 이내 처리

5. 개인정보의 안전성 확보 조치

  • 암호화: 비밀번호는 bcrypt 해시, 인스타그램 액세스 토큰은 Fernet(AES) 암호화 저장
  • 접근 통제: 관리자 계정 2FA, DB 직접 접근은 허용된 IP만
  • 통신: 모든 API 통신 HTTPS(TLS 1.2+)
  • 멀티테넌시 격리: 모든 쿼리에 user_id 필터 강제 — 다른 샵의 데이터 노출 불가 (코드 레벨 검증)
  • 로그: 개인정보 접근·변경 이력 90일 보관

6. 쿠키 및 자동 수집 도구

서비스는 제3자 광고·추적 쿠키를 사용하지 않습니다. 로그인 유지 목적의 자체 세션 쿠키(또는 localStorage 토큰)만 사용하며, 브라우저 설정에서 삭제 가능합니다.
