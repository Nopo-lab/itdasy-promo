회원가입 시 "개인정보 국외 이전에 동의합니다" 체크박스를 통해 동의를 받습니다. 미동의 시 서비스 제공이 불가능하며, 추후 설정 → 계정 → 국외이전 동의 철회에서 동의를 철회할 수 있습니다. 철회 시 Gemini 기반 AI 기능(캡션·비서)은 사용 불가합니다.

2-3. 가명처리(Pseudonymization)

AI 분석 시 고객 개인정보는 서버에서 가명(예: "고객#1", "고객#2")으로 치환된 후 외부 AI 서비스에 전달됩니다. 실제 이름·전화번호 등 식별 정보는 Google 등 외부 서버에 저장되지 않습니다. Google Gemini API 정책상 입력 데이터는 모델 학습에 사용되지 않습니다.

4. 정보주체의 권리·의무 및 행사 방법

이용자(회원·고객)는 「개인정보 보호법」 제4조·제35조~제37조에 따라 언제든지 다음 권리를 행사할 수 있습니다:

  • 열람권: 처리하는 개인정보의 내용·처리 목적·제3자 제공 내역 조회
  • 정정·삭제권: 본인의 정보가 사실과 다를 경우 정정 요구, 수집·처리 목적 달성 시 삭제 요구
  • 처리 정지권: 특정 목적의 처리(예: 마케팅, AI 분석)에 대해 정지 요구
  • 동의 철회권: 선택적 동의 항목(마케팅 수신 등)은 언제든 철회 가능. 국외 이전 동의 철회 시 해당 AI 기능 불가
  • 이동권(데이터 포터빌리티): 앱 내 설정 → 계정 → 데이터 내보내기 → 고객·예약·매출 CSV 즉시 다운로드
  • 회원 탈퇴권(완전 삭제): 앱 내 설정 → 계정 탈퇴 버튼 1회 → "탈퇴" 두 글자 입력 확인 → 즉시 모든 개인정보 영구 삭제 (Apple/Google 앱스토어 가이드라인 의무 준수, 웹 폼이나 이메일 요청 필요 없음)
  • 자동화된 의사결정 거부권: 제6조의2 참고

위 권리 행사는 앱 내에서 즉시 가능하며, 이메일(contact@itdasy.com) 요청 시 10일 이내 처리합니다. 대리인을 통한 행사는 위임장·신분증 사본 첨부로 가능합니다. 회사는 열람·정정·삭제 요구에 대해 정당한 사유 없이 거부·지연하지 않으며, 거부 시 사유와 불복 방법을 서면 통지합니다.

📲 만 14세 미만 아동의 경우 법정대리인이 대신 권리를 행사할 수 있습니다.

5. 개인정보의 안전성 확보 조치

  • 암호화: 비밀번호는 bcrypt 해시, 인스타그램 액세스 토큰은 Fernet(AES) 암호화 저장
  • 접근 통제: 관리자 계정 2FA, DB 직접 접근은 허용된 IP만
  • 통신: 모든 API 통신 HTTPS(TLS 1.2+)
  • 멀티테넌시 격리: 모든 쿼리에 user_id 필터 강제 — 다른 샵의 데이터 노출 불가 (코드 레벨 검증)
  • 로그: 개인정보 접근·변경 이력 90일 보관

6-의2. 자동화된 의사결정 (AI 처리 고지)

본 서비스는 다음 기능에서 인공지능(Google Gemini API)을 활용한 자동화된 처리를 수행합니다. 이는 단순 보조·추천이며, 법률적·재정적 효과를 직접 발생시키는 결정(예: 자동 결제, 자동 해지)은 하지 않습니다.

  • AI 캡션/해시태그 생성: 사용자가 입력한 시술 정보를 바탕으로 소셜미디어 문구 제안 (최종 게시 여부는 사용자가 결정)
  • AI 비서(챗봇): 예약·매출 데이터 기반 조회·초안 생성 (실제 DB 변경은 사용자 확인 후 실행)
  • 이탈 위험 고객 추정: 방문 간격 통계 기반 점수 계산 (알림 발송 결정은 사용자 몫)
  • 가명처리(Pseudonymization): 모든 AI 호출 전 고객 이름은 "고객#1" 등으로 치환되어 외부 전송 시 식별 불가

이용자는 언제든 AI 기능 사용을 거부할 수 있으며(앱 내 설정 → 국외이전 동의 철회), 거부 시 AI 기능은 제한되나 핵심 서비스(고객·예약·매출 관리)는 정상 이용 가능합니다. AI 결과가 부정확·부적절하다고 판단될 경우 앱 내 🚩 신고 버튼으로 접수 시 24시간 내 검토·처리합니다.
